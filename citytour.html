City Tours

Itinerary Creator Application
MVP Full Stack Application

Description

Itinerary creator application to help users to create itineraries based on their starting locations. It allows users to search landmarks based on state and city, add them to their itinerary, generate travel routes, and see details of landmarks. Users also can rate landmarks and leave reviews.

Technologies used

  • Java
  • JavaScript
  • React
  • Redux
  • HTML
  • CSS
  • Spring Boot
  • PostgreSQL
  • RESTful API

Leading a team as a Scrum Master

As Scrum Master, I led a team of 4 by managing meeting schedules, assigning tasks, tracking daily and weekly progress, and supporting team members as needed to complete the project. Our team achieved a 9/10 score for the demo and presentation.

My contributions

  • Created Wireframe
  • Create Itinerary : An itinerary includes a name, a starting point, date of itinerary
  • Add Landmark to Itinerary : Users can add landmarks to their itinerary so that they are included in my travel route
  • Delete Itinerary : Users have the ability to delete a travel itinerary
  • Rate Thumbs Up / Thumbs Down : Users can thumbs up or thumbs down a landmark so that it can help show its popularity to other visitors in the region
  • Submit Landmark Review : Users can submit feedback for a landmark so that others may read it before deciding to visit that location A review includes: title, description

Vendo-Matic-800

Virtual Vending Machine Application

Description

Virtual vending machine application with a command-line interface allowing users to deposit money, choose a product, and return the correct change. Inventory is loaded via a text file and transactions are also logged into a text file.

Softwares used

  • Java
  • OOP
  • File I/O
  • RESTful API
  • JUnit

  • Programmed classes for log, purchase, sales report, and tests
  • Developed model, service, controller, DAO for Transfer/TransferStatus/TransferType/Account classes
  • Created exception classes on the client side and the server side
